Export & Deploy to GitHub Pages

Download the generated files and push them to your GitHub repo to host on GitHub Pages. Each card gets its own URL: https://YOUR-ORG.github.io/REPO/cards/CARD-ID.html

Repository structure

Quick deploy: Download each card's HTML, push to cards/ in your repo, enable GitHub Pages (Settings → Pages → branch: main / root). Each card is a self-contained file — fonts included, no build step.
